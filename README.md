# Tech Reader

**帮助开发者摆脱翻译依赖，真正读懂英文技术文档。**

核心理念：越用越不需要它。

## 功能

- **句型分析**：选中任意英文句子，AST 树状拆解骨架 + 修饰层 + 固定短语，理解句子结构
- **Hover 查词**：停留在单词上自动展示中文/英文释义，记录到个人词库
- **生词复习**：超过 3 天没见过的词，下次 hover 进入挑战模式，自己想起来才算掌握
- **词库可视化**：点击图标打开词库，查看收录单词、掌握进度、查询统计

## 安装

1. 下载 `tech-reader.zip` 解压
2. Firefox 地址栏输入 `about:debugging`
3. 点「此 Firefox」→「临时加载附加组件」→ 选择 `manifest.json`

正式版本发布后可直接从 Firefox Add-ons 安装。

## 配置

点击工具栏图标打开词库，右上角进入设置，填入：

- **API Key**：支持 OpenAI、DeepSeek、通义千问、月之暗面等任意 OpenAI 兼容接口
- **Base URL**：对应服务的接口地址
- **模型**：推荐 `deepseek-chat`（便宜且效果好）

## 词义语言

设置页可切换中文/英文解释，默认中文，适合入门。熟悉后切换英文模式建立直接的英文→概念连接。

## 掌握度机制

```
第一次 hover → 记录生词
3天后再遇到 → 挑战模式，先不展示释义
  ├─ 自己想起来，直接移走 → 复习通过 +1
  └─ 点「看答案」→ 还没掌握，继续记录
复习通过 2 次 → 标记为「已掌握」
```

## 技术栈

Firefox WebExtension MV2 · IndexedDB-free · browser.storage.local · OpenAI 兼容 API
